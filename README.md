# Help Desk

Учебный проект на Java 17, Spring Boot, Spring MVC и Thymeleaf.

## Возможности

- главная страница сайта
- страница "О нас"
- страница "Контакты поддержки"
- страница FAQ
- блок "Что указать в обращении"

## Запуск

1. Установить JDK 17.
2. Запустить проект:

   ```powershell
   .\mvnw.cmd spring-boot:run
   ```

3. Открыть в браузере:

   ```text
   http://localhost:8080
   ```

## Проверка страниц

- `http://localhost:8080/`
- `http://localhost:8080/about`

- `http://localhost:8080/faq`
